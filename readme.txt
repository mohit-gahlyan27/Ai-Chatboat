# AI Chatbot using Flask and Groq API

## Project Overview

This project is a simple AI Chatbot web application developed using Python Flask, HTML, CSS, and Groq API.  
The chatbot allows users to ask questions through a web interface and receive AI-generated responses instantly.

The project demonstrates the integration of frontend and backend technologies with Artificial Intelligence APIs.

---

# Technologies Used

## 1. Python
Python is used as the main backend programming language.

### Purpose:
- Handles server-side logic
- Connects frontend with AI API
- Processes user input and responses

---

## 2. Flask Framework
Flask is a lightweight Python web framework.

### Purpose:
- Creates web server
- Handles routes and requests
- Connects HTML pages with Python backend

### Main Features Used:
- Routing
- Templates
- Form handling

---

## 3. HTML
HTML is used to design the structure of the chatbot webpage.

### Components Used:
- Form
- Input box
- Button
- Reply section

---

## 4. CSS
CSS is used to style the chatbot interface.

### Purpose:
- Improve UI design
- Add colors and spacing
- Create modern chatbot appearance
- Make interface responsive and attractive

---

## 5. Groq API
Groq API is used to generate AI responses.

### Purpose:
- Sends user message to AI model
- Receives intelligent response
- Provides fast AI inference
